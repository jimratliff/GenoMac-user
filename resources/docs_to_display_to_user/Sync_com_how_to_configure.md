# How to configure the Sync.com desktop app

> [!NOTE]
> This assumes the Sync app is version-2.x, which allows arbritrary sync-folder location (as opposed to version-5.x beta, which is compliant with Apple’s File Provider API).

## Launch Sync.com
- ❑ Make Sync.com the active app. (The Hypervisor should have launched Sync.com for you already.)
- You’ll see a “Sync Setup” dialog box: “Welcome to Sync! You’re installing version 2.xxx of the Sync desktop app. Keep your files safe, secure, and in sync across all your devices. Click Next to sign in or create a new account.”
- ❑ Click the “Next>” button.
- The default browser will open a “Sign in” page (URL is https://cp.sync.com/login-webauth…)
- ❑ Enter the email address for your Sync.com account into the text box.
- ❑ Click “Sign in”

## Specify location of sync folder
- The browser will open another “Sync Setup” window: “Sync Folder Location”.
- “During installation a Sync folder is placed on your computer. Files you place inside the Sync folder will be backed up to the cloud and synced to any other computers you’ve installed Sync on.”
- There are two text entry fields:
- Sync folder location: ~/
- Device name:
- ❑ Click “Next>” button.
- A new “Sync Setup” window opens: “Sync complete”: “Sync has been successfully set up on this computer. Click finish to exit the installer.
- ❑ Click the “Finish>” button.

## Select the subfolders to sync
- Click on the Sync icon in the menubar
- ❑ In the drop-down dialog box, click on the gear (⚙️) icon in the upper right.
- ❑ Choose “Preferences”
- ❑ Choose the “Advanced” tab.
- ❑ Selectively choose the folders you want to sync.
- ❑ Click the “Apply” button.

> [!WARNING]
>
> It can take a few minutes before syncing actually begins. Be patient!

## Return to terminal and acknowledge
- ❑ Type `done` to acknowledge that you’ve completed these manual steps.
