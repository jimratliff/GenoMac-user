# How to configure Touch ID

## Find the already-opened Touch ID & Password panel of System Settings
- Locate the Touch ID & Password panel from System Settings that has already been opened for you.
- Click on the “⊕” button that is annotated “Add Fingerprint”
- Follow the instructions, **being sure to use the particular finger** you’ve been directed to use.

## Return to terminal and acknowledge
- [ ] Type `done` to acknowledge that you’ve completed these manual steps.
