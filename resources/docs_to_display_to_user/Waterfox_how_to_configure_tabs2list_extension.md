# How to configure the Tabs2List extension for the Waterfox web browser

## Make Waterfox active and open the list of installed extensions
- ❑ Make Waterfox active
  - The script will already have opened the Waterfox application for you.
- ❑ From the application menu, choose Tools » Extensions and Themes (⇧⌘A)

## Enable Tabs2List
- Tabs2List
  - ❑ Toggle the switch to ON ✅
 
## Open the preferences for Consent-O-Matic
- ❑ Context-click on the “…” button to the right of the switch you just toggled, and select “Preferences”

## Selectively change the following default settings
- “Color scheme”
  - ❑ Change to “Camp Firefox”
- “Font size”
  - ❑ Change to 9

## Return to the terminal
Now return to the terminal and acknowledge you have completed the configuration of the Enhancer for YouTube
extension for Waterfox.
