# How to configure keyboard shortcuts for extensions for the Waterfox web browser

## Make Waterfox active and navigate to the Manage Extension Shortcuts page
- ❑ Make Waterfox active
  - The script will already have opened the Waterfox application for you.
- ❑ From the application menu, choose Tools » Extensions and Themes (⇧⌘A)
- ❑ Click-and-hold the gear (⚙️) icon in the upper-right corner of the page and select “Manage Extension Shortcuts”
  - The “Manage Extension Shortcuts” page will be displayed
 
## For each of the following extensions, make the indicated change in the assigned shortcut
- In some cases, the assigned shortcut will be removed entirely.
- In other cases, the shortcut assigned to an action will be replaced with a different shortcut.
### 1Password - Password Manager
- “Open the extension”
  - ❑ Replace ⌘. → F6
- “Lock the extension”
  - ❑ REMOVE the ⇧⌘L shortcut
### Allow Right-Click (no changes)
- “Execute Action”
  - NO CHANGE, because no shortcut is assigned
### Downie (remove shortcut)
- “ Open current site with Downie
  - ❑ REMOVE the ⇧⌃D shortcut
### Enhancer for YouTube™ (no changes
Although there are 38 entries, they all default to ❌. No change necessary.
### Manage My Tabs - Tab Manager (replace shortcut)
- “Activate toolbar button”
  - ❑ Replace ⌥M → ⇧⌃M
### Raindrop.io (replace one shortcut)
- “Open sidebar”
  - ❑ Replace ⌘. → F4
- “Bookmark/Save highlights”
  - NO CHANGE: Remains at ⌃X (at least for now, since this isn’t a familiar workflow)
- All the remaining items default to ❌, which is fine. No change necessary
### Tabs2List (replace one shortcut)
- “Toggle the sidebar”
  - ❑ Replace ⌥1 → F2
- “Activate toolbar button”
  - NO CHANGE: Remains at ⌥2
### Tree Style Tab (keep 5, remove 7, and leave untouched 42 more that default to ❌)
- DO NOT CHANGE the following 5 entries
  - “Toggle ‘Tree Style Tab’ Sidebar”
    - F1
  - “Simulate Up Key on Tree”
    - ⇧⌃↑
  - “Simulate Down Key on Tree”
    - ⇧⌃↓
  - “Simulate Left Key on Tree”
    - ⇧⌃←
  - “Simulate Right Key on Tree”
    - ⇧⌃→
- REMOVE THE SHORTCUT from each of the remaining 7 entries that have a shortcut
  - “Scroll Tabs Up by Lines”
    - ❑ REMOVE ⌥↑
  - “Scroll Tabs Up by Page"
    - ❑ REMOVE ⌥PageUp
  - “Scroll Tabs to Top"
    - ❑ REMOVE ⇧⌥Home
  - “Scroll Tabs Down by Lines”"
    - ❑ REMOVE ⌥↓
  - “Scroll Tabs Down by Page""
    - ❑ REMOVE ⌥PageDown
  - “Scroll Tabs Up by Lines”"
    - ❑ REMOVE ⌥PageUp
  - “Scroll Tabs to End""
    - ❑ REMOVE ⌥End
  - “Toggle Sub Panel”"
    - ❑ REMOVE F2
### Chessvision: No shortcuts
### Consent-O-Matic: No shortcuts
### Markdown Viewer: No shortcuts

## Return to the terminal
Now return to the terminal and acknowledge you have completed the installation of the “Activist - Balanced”
theme for Waterfox.
