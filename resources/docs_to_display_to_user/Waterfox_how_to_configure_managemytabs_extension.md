# How to configure the Manage My Tabs extension for the Waterfox web browser

## Make Waterfox active and open the list of installed extensions
- ❑ Make Waterfox active
  - The script will already have opened the Waterfox application for you.
- ❑ From the application menu, choose Tools » Extensions and Themes (⇧⌘A)

## Enable Manage My Tabs - Tab Manager
- Manage My Tabs - Tab Manager
  - ❑ Toggle the switch to ON ✅
    - ❑ Pin 📌 ? ✅
 
## Open the Settings for Manage My Tabs
- ❑ In the toolbar of a Waterfox window, locate the pinned “Manage My Tabs” icon
- ❑ Click on the pinned “Manage My Tabs” icon in the toolbar
    - This will show a “Manage My Tabs” page
- ❑ Click on the gear (⚙️) in the upper-left corner of the “Manage My Tabs” page
    - This will reveal a sidebar on the left containing Settings for the “Manage My Tabs” extension 

## Selectively change the following default settings
- “Columns”
  - ❑ Change to 1
- “Display number of tabs”
  - ❑ Toggle OFF (❌)
- “Limit the number of tags in a tab group”
  - ❑ Toggle OFF (❌)

## Return to the terminal
Now return to the terminal and acknowledge you have completed the configuration of the Manage My Tabs
extension for Waterfox.
