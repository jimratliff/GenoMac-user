# How to enable the Witch preference pane

## Make the Witch preference pane active
- ❑ Make the Witch preference pane active
  - The script will already have opened the Witch preference pane for you.
- Alternatively:
  - ❑ Open System Settings (⌘,)
  - ❑ At the bottom of the left sidebar, click on “Witch”

## Enable Witch
- ❑ Toggle ON the switch to “Enable Witch” in the upper-left corner of the right-hand pane of System Settings.

## Return to the terminal
Now return to the terminal and acknowledge you have completed enabling the Witch preference pane.
