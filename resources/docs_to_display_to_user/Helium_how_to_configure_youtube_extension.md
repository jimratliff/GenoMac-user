# How to configure the Enhancer for YouTube™ extension for the Helium web browser

## Make Helium active and open the list of installed extensions
- ❑ Make Helium active
  - The script will already have opened the Helium application for you.
- ❑ From the application menu, choose Window » Extensions (or click the jigsaw-puzzle icon in the toolbar and then “Manage Extensions”)

## Enable “Enhancer for YouTube™” and open its preferences
- ❑ Find “Enhancer for YouTube™” in the list of installed extensions
- ❑ Toggle ON the switch to enable Enhancer for YouTube™”
- Either:
  - Navigate methodolodically to the setting page by:
    - ❑ Click on the “Details” button associated with “Enhancer for YouTube™”, which opens a new page about “Enhancer for YouTube™”
    - ❑ Click on the “share-icon” icon on the “Extension options” row
  - Go directly to: chrome-extension://ponfpcnoihfmfllpaingbgckeeldkhle/html/options.html

## Selectively change the following settings from their default values
### “Autoplay”
- “Disable autoplay”
  - ❑ Turn ON ✅
- “Prevent videos loaded in background tabs from automatically playing as well”
  - “Prevent videos loaded in the foreground tab from automatically playing as well”
    - ❑ Turn ON ✅
### “Controls”
Note: Each control is represented by an icon that I can’t replicate here. Instead, I’ll refer to
each icon by a position and text that you’d see if you hover over the icon.

- Row 1 of controls:
  - Item 1: Loop mode (“Click or right-click to toggle the loop mode”)
    - ❑ Turn OFF ❌ this item
  - Item 4: Volume booster (“Click to toggle the volume booster”)
    - ❑ Turn OFF ❌ this item
  - Item 10: Use mouse wheel to control playback speed (“Use the mouse wheel to control
    the playback speed or click to select the default playback speed or right-click to
    select the normal playback speed”)
    - ❑ Turn OFF ❌ this item
  - Item 12: Video filters (“Click to toggle the video filters or right-click to toggle
    the video filters panel visibility”)
    - ❑ Turn OFF ❌ this item


## Return to the terminal
Now return to the terminal and acknowledge you have completed the configuration of the Enhancer for YouTube
extension for Waterfox.
