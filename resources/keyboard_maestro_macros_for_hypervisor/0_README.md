# Two Keyboard Maestro macros used by the Hypervisor

As explained in the [main README](https://github.com/jimratliff/GenoMac-user/blob/main/README.md), the Hypervisor assumes that you will use a pair of Keyboard Maestro macros to (a) enter the license for Alfred’s Powerpack and (b) enter the serial number for Keyboard Maestro itself, respectively.

A redacted version of these macros is provided here, for your convenience, at `resources/keyboard_maestro_macros_for_hypervisor/GenoMac Bootstrap Macros.kmmacros`. In other words, there are placeholders where the two license credentials need to be. You can replace those placeholders with your own credentials.
